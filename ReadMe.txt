Interview Technical Assessment 


Using the data files provided "SampleData.csv" and "DepartmentList.csv" aggregate the data. 

The result should be a summary of calls that occurred within each 15-minute period (interval)
grouping by interval and department, display a sum of received and answered calls with the total wait and talk time for each interval. 
See file "ExpectedOutput.csv" for reference. 

This exercise can be developed in any language or framework you are familiar with (c# .NET, VB .NET, Java, SQL, etc...)
